KSM POS System

The is the first group work where we decide to built a "touch" screen Point of sale app component implemented with React/Redux and Node.js(backend).
This is a rough demo POS for shops and small business to build their mobile storefront. It features a list of items in inventory, a list of items on screen, checkout and edit, and pay confirmation page. A total sale list is saved in user's local storage.
UI and Click animation for lists and form input are inspired by current Tablet usage. Backend sales report screen.
